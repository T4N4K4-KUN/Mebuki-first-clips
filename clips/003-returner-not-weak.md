# 003 — 昔やめた人は、弱い人じゃない

status: idea
format: 9:16 vertical / 20〜35秒 / 字幕あり

## Purpose

返り咲き層の自己否定を下げる。過去に離れたことを弱さではなく、戻るための経験として扱う。

## Viewer state

- 昔やったが続かなかった。
- もう遅いと思っている。
- 最新環境についていけないと思っている。

## Core message

昔やめた人は、弱い人じゃない。

## Desired action

以前の自分を責めず、今日の小さな再開地点を1つ決める。

## Draft constraints

- メンタル系ショートの言葉をそのまま使わない。

3. 今はAIも、ログも、戻る道もある。
4. 今日の勝ち：前回どこで止まったかを書く。
