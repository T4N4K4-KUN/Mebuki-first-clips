# AUDIENCE_IDEA_RESEARCH_WOMEN_NON_GEEK

001の題材選びのための調査メモ。

目的は「女性向け」を雑に決めることではない。
ギーク層の直感だけで `インベーダー風ゲーム` に寄せすぎないため、女性を含む非ギーク層が自分ごと化しやすい入口を探す。

checked_date: 2026-06-15

## Evidence Status

このメモは、以下を分けて扱う。

- Data-backed: 公開調査・市場データ・ランキング等で根拠がある。
- Inference: データからの推論。題材選定の仮説。
- Rejected / weak: 直感では良さそうだが、001の題材として根拠や実装難易度に問題がある。

重要:

- 「女性が何を作りたいか」を直接測った信頼できるデータは、現時点では未確認。
- 使えるのは、ゲームジャンル嗜好、SNS利用、推し活/創作行動、既存ヒット例からの間接推定。
- したがって、001の題材は「女性向け」と断定せず、「非ギーク層にも入口がある題材」として扱う。

## Caution

女性は一枚岩ではない。

年齢、生活文脈、SNS利用、ゲーム経験、推し活、創作経験、仕事/学習目的でかなり違う。

したがって、001の題材を「女性ならこれ」と決めるのは危険。
代わりに、複数の入口を用意して、視聴者が「これなら私も作ってみたい」と思える余地を作る。

## Research Signals

## Evidence-Backed Facts

### F1. 女性ゲームユーザー比率はジャンルで大きく違う

Quantic Foundry は、27万人超のゲーマー調査をもとに、ジャンル別の女性比率を分析している。

根拠:

- Match 3 は、そのジャンルに言及したゲーマーの 69% が女性。
- Match 3 と Family/Farm Sim は、調査対象ジャンルの中で女性比率が最も高い。
- ジャンル平均の女性比率は 2% から約70%まで大きくばらつく。
- ただし、この69%は「女性の69%がMatch 3を遊ぶ」という意味ではない。

Source:
- Quantic Foundry, "Beyond 50/50: Breaking Down The Percentage of Female Gamers by Genre"  
  https://quanticfoundry.com/2017/01/19/female-gamers-by-genre/

001への意味:

- 「インベーダー風」だけで広く刺さるとは言えない。
- データ上は、Match 3 / Family-Farm Sim に近い題材の方が、女性比率の高い既存ジャンルに近い。
- `スイカゲーム系` は、落ちもの・合体・パズル寄りで、この方向に近い。

### F2. 日本ではSNS・動画利用が広く、Instagramは女性比率が高い

DataReportal Digital 2025 Japan によると、日本では2025年1月時点で:

- インターネットユーザー: 1億900万人、人口比88.2%。
- ソーシャルメディア user identities: 9700万、人口比78.6%。
- Instagram: 5750万人、成人広告オーディエンスの57.8%が女性。
- TikTok: 18歳以上で2690万人、成人広告オーディエンスの52.1%が女性。
- YouTube: 7870万人、成人広告オーディエンスの48.4%が女性。

Source:
- DataReportal, "Digital 2025: Japan"  
  https://datareportal.com/reports/digital-2025-japan

001への意味:

- 写真・動画・SNS文脈は、女性を含む広い層に届く可能性がある。
- ただし「SNSを使う」ことは「SNS向けアプリを作りたい」ことを直接意味しない。
- `推し色カードメーカー` のようなデザイン系は、利用文脈としてはあり得るが、001の実演題材としては別途難易度評価が必要。

### F3. 推し活は実在する大きな文化だが、カードメーカーの1時間制作適性は別問題

推し活は日本の若年層・女性を含むファン文化として存在する。
ただし、推し活向けカード/グッズは、デザイン精度・余白・フォント・色味・素材配置へのこだわりが強くなりやすい。

Source:
- Oshikatsu overview  
  https://en.wikipedia.org/wiki/Oshikatsu
- 実例動画候補:
  - 推し活自己紹介カード: https://youtu.be/JFl89KXm3Cg
  - 推しプリクラ風カード: https://youtu.be/RvwmoFzFiFw
  - 推しカード作成: https://youtu.be/33zYT6FyURU

001への意味:

- 推し活/カード系は「作りたい人」はいそうだが、1時間の初心者ツカミには危険。
- 理由は、仕様がロジックではなく美的微調整に寄るため。
- mm単位の調整、フォント、余白、色、写真切り抜きが発生する。
- 001では本命から外す。

### F4. スイカゲームは実在ヒットで、ルールが単純なカジュアルパズル

Suika Game は、果物を落として同じ果物を合体させるパズルゲーム。
日本発で、Nintendo Switch版・モバイル版などで広く知られた。

Source:
- Suika Game overview  
  https://en.wikipedia.org/wiki/Suika_Game

001への意味:

- 「スイカゲーム系」は、実在ヒットの文脈がある。
- ルールが説明しやすく、見た目も柔らかい。
- ただし物理挙動・合体判定・ゲームオーバー判定は、実装が重くなり得る。
- 1時間教材では、完全再現ではなく「スイカゲームみたいな落ちもの合体ミニ版」にする必要がある。

### 1. Mobile/casual game affinity

Data-backed:

- Quantic Foundryの調査では、Match 3 と Family/Farm Sim は女性比率が高いジャンル。
- Tactical Shooter や Sports は女性比率が低いジャンルとして出ている。
- つまり、ゲーム一般ではなくジャンル差が非常に大きい。

Inference:

- 001の非ギーク入口としては、競争・反射神経・攻撃より、パズル、育成、収集、柔らかい見た目の方が安全。

### 3. Suika Game type

`Suika Game` は、ギーク向けアクションではなく、見た目が柔らかい物理パズル。

001に使える要素:

- ルールが一瞬で分かる。
- 見た目が怖くない。
- 果物・丸い形・合体という直感性がある。
- 失敗しても笑える。
- 「自分版」に置き換えやすい。

置き換え例:

- フルーツ合体
- もち/団子合体
- コスメ合体
- 猫クッション合体
- 推しカラーの宝石合体

## Implication For 001

`インベーダー風ゲーム` はギーク/返り咲き層には強い。
ただし、女性を含む非ギーク層全体のツカミとしては狭い可能性がある。

001の構造はそのままで、題材を複数見せる方がよい。

```text
最初の1時間で、ゲームは動く。

たとえば、
- インベーダー風
- スイカゲーム系
- 好きなキャラでSnaky
- カフェ注文ミニゲーム
- 献立ルーレット
```

この言い方なら、ゲーム好きだけでなく、日常ツールやカジュアルゲームに関心がある層も入れる。

## Recommended First Clip Hook

001の最初に見せる題材は、ひとつに決めず、3つを高速に見せる。

```text
AIに頼むと、最初の1時間でここまで動く。

1. インベーダー風ゲーム
2. スイカゲーム系ミニ版
3. 好きなキャラでSnaky
```

その後で、

```text
今日はこの中から、いちばん作り方が見えやすいミニゲームを作る。
```

とすると、ギーク層も非ギーク層も残りやすい。

## Candidate Prompt Ideas

### Suika-like Merge

```text
かわいいフルーツが上から落ちてきて、
同じフルーツがぶつかると一段大きなフルーツに合体するゲームを作りたい。
最初はスコアとゲームオーバーだけでよい。
```

### Character Snaky

```text
好きなキャラを動かして、アイテムを集めるたびに少しずつ長くなるゲームを作りたい。
壁や自分の体にぶつかるとゲームオーバー。
キャラやアイテムは後から差し替えられるようにしたい。
```

### Kawaii Card Maker

```text
名前と好きな色を入れると、
推し色のかわいいプロフィールカードを作るアプリを作りたい。
画像として保存できる機能は後でよい。
まず画面にカードが表示されればよい。
```

Status:

- 001本命からは外す。
- 理由はデザイン要求の精度が高く、1時間の初心者教材では沼る可能性が高い。

### Cafe Order Game

お客さんの注文に合わせて、

## Current Judgment

001で `ゲーム` と言うなら、インベーダー風だけにしない。

より広くするなら:

```text
最初の1時間で、自分の好きなミニアプリが動く。
ゲームでも、カードメーカーでも、写真アプリでもいい。
```

ただし、実演題材としては、ロジックが見えやすい `スイカゲーム系ミニ版` または `好きなキャラでSnaky` が強い。

理由:

- 見た目が柔らかい。
- ルールが一瞬で分かる。
- スコア/ゲームオーバー/改造が分かりやすい。
- ギーク層にも非ギーク層にも説明しやすい。

Current ranking:

```text
本命: スイカゲーム系ミニ版
対抗: 好きなキャラでSnaky
ギーク/返り咲き向け: インベーダー風
保留: 献立ルーレット、カフェ注文ミニゲーム
001本命から外す: 推し色カードメーカー、猫のおやつキャッチ
```

Rationale:

- スイカゲーム系: 既存ヒットがあり、カジュアルパズル文脈に乗る。
- 好きなキャラでSnaky: Snaky Catのように、既存の強いゲーム型をキャラ置換できる。
- インベーダー風: 返り咲き層には強いが、非ギーク層全体には狭い。
- 推し色カードメーカー: 需要はあり得るが、1時間初回ではデザイン調整が主戦場になる。
- 猫のおやつキャッチ: かわいいが、実在人気ジャンルとしての裏取りが弱い。
