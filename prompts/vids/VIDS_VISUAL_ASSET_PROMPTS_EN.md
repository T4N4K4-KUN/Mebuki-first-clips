# Vids Visual Asset Prompts — Scene-ID Asset Briefs

Use this file to create external visual assets for Vids.
Each asset prompt includes the scene ID used in `VIDS_SCENE_PROMPTS_EN.md`.

Important:

- These are prompts for still visual assets: icons, illustrations, cards, UI mockups, and game-like visual parts.
- Do not generate final video scenes here. Generate clean source visuals that Vids can place, crop, animate, or combine.
- Name the real reference game inside the asset prompt so the external artist or image tool understands the target.
- Keep the visual close enough that viewers can understand the game type at a glance, but do not copy protected expression.
- Do not use original game screenshots, logos, title screens, characters, UI layouts, fonts, sound marks, item designs, or exact color schemes.
- The first audience is Japanese. Any viewer-facing text inside the artwork should be Japanese, except very simple English product names such as "Codex app" and "Mebuki".
- Keep the scene ID visible in the file name or metadata, not inside the artwork unless requested.

## Common Prompt — Vertical 9:16 Assets

Use this before any scene-specific prompt when creating vertical assets.

```text
Create original visual assets for a vertical 9:16 Vids educational video. Canvas: 1080 x 1920. Keep all important visual elements inside the central safe area. Leave generous negative space near the top and bottom for captions. Use a calm, practical, beginner-friendly visual style: dark neutral background, Japanese viewer-facing text when text is needed, subtle green and cyan highlights, clean cards, simple geometric shapes, and soft but not childish UI. When a real game is named as reference, keep the core gameplay silhouette recognizable enough that viewers understand the type, but redesign the protected expression: no screenshots, logos, title screens, exact characters, exact UI, exact board layout, exact item designs, exact fonts, or exact color palette. Use the real game name only as production reference text, not as artwork text.
```
